# Production Art Workspace

Prompt 15 defines the art system; Prompt 16 will populate the runtime asset folders in batches of no more than eight separately reviewed images.

- `reference-sheet.html` — visual summary of the approved direction
- `style-tokens.css` — exact reference-sheet palette and typography tokens
- `asset-manifest.json` — complete approved filenames and dimensions
- `../docs/ART_BIBLE_v1.md` — authoritative production rules

Do not place source masters in the itch.io build. Runtime art will live under `assets/art/`; editable masters and generation records belong in the development archive.
